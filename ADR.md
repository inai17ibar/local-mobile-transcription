# ADR — Architecture Decision Records

アーキテクチャ決定の台帳です。重要な技術的決定とその根拠を記録します。

---

## 記録テンプレート

新しい決定を記録する際は、以下のテンプレートをコピーして追記してください。

```markdown
## ADR-{番号}: {タイトル}

- **日付**: YYYY-MM-DD
- **状態**: 提案中 / 承認済み / 却下 / 廃止
- **決定者**: {名前 or チーム}

### コンテキスト
{なぜこの決定が必要になったか。背景・制約を記述。}

### 検討した選択肢

| 選択肢 | メリット | デメリット |
|--------|---------|-----------|
| A: {選択肢A} | ... | ... |
| B: {選択肢B} | ... | ... |

### 決定
{選択した選択肢と理由}

### 影響
{この決定が与える影響・トレードオフ}
```
